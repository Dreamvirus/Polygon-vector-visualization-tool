# 多边形向量可视化工具

一个基于Web的3D向量多边形可视化工具，可以将向量数组转换为可视化的封闭多边形图形。

![项目预览](https://img.shields.io/badge/status-active-success.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 功能特点

- ✨ **向量解析**：自动解析 `X=... Y=... Z=...` 格式的3D向量数组
- 🎨 **多颜色绘制**：支持添加多个多边形，每个使用不同颜色区分
- 📐 **智能缩放**：自动计算边界并缩放，确保所有图形完整显示
- 🖱️ **交互式管理**：可单独删除或清除所有多边形
- 📱 **响应式设计**：适配桌面和移动设备
- ⚡ **本地运行**：使用Vite快速开发服务器，即时热更新

## 快速开始

- Node.js >= 14.0.0
- npm >= 6.0.0

1. 克隆项目到本地

```bash

cd polygon-visualizer
```

2. 安装依赖

```bash

```

3. 启动开发服务器


4. 在浏览器中打开显示的本地地址（通常是 `http://localhost:5173`）

## 使用说明

### 向量数组格式

输入的向量数组需要使用JSON格式，每个向量字符串包含X、Y、Z坐标（Z轴值会被忽略）：

```json
[
    "X=1500.313 Y=589.756 Z=0.000",
    "X=1500.313 Y=306.219 Z=0.000",
    "X=1296.167 Y=306.219 Z=0.000",
    "X=1296.167 Y=257.613 Z=0.000"
]
```

### 操作步骤

1. **输入向量数组**：在文本框中粘贴或输入向量数组
2. **添加多边形**：点击"添加多边形"按钮（或使用 `Ctrl + Enter` 快捷键）
3. **查看结果**：多边形会显示在画布中心，自动使用不同颜色
4. **管理多边形**：
   - 在下方列表中查看所有已添加的多边形
   - 点击"删除"按钮移除单个多边形
   - 点击"清除所有"按钮移除所有多边形

## 常见问题

### Q: 为什么我的多边形没有显示？

A: 请检查：
1. 向量数组格式是否正确（需要是有效的JSON格式）
2. 是否至少有3个顶点
3. 坐标值是否有效（不能是NaN或Infinity）

### Q: 如何调整多边形的显示大小？

A: 程序会自动计算所有多边形的边界并进行缩放，确保所有图形都能完整显示在画布中。

### Q: 支持多少个多边形？

A: 理论上没有限制，但过多的多边形可能会影响渲染性能。预设了10种颜色循环使用。
